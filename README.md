# propertyPortal

A website which allows user to buy, rent and sell real estate properties.  

## Overview

**propertyPortal** is a application that displays property listings with details (images, price, location, description). Data is stored in a JSON file and rendered dynamically in the browser (an implementation of server). The project uses plain HTML, CSS, and JavaScript and JQuery.

## Features

- Property list rendering from `properties.json`
- Search and filter by city, type, price range
- Property detail cards with images and key attributes
- Responsive layout for desktop and mobile

## Tech Stack

- HTML (structure)
- CSS (styles)
- JavaScript, JQuery (client-side logic) 
- JSON (property dataset)

## Project Structure

- css/ Stylesheets
- html/ HTML pages
- js/ JavaScript modules
- images/ Property and UI images
- properties.json Dataset of properties
- homePage.html Entry point

## Usage

1. Clone repository  
   `git clone https://github.com/JonZylfiu/propertyPortal.git`

2. Open `index.html` in a browser.

3. Use search bar and filters to explore listings.

## Development

- Property data is defined in `properties.json`.
- Client-side logic is implemented in the `js/` directory.
- Layout and responsiveness are handled in `css/`.
- Open `homePage.html` directly in a browser.
